# 入力 Excel の形式と終端判定

## ファイル名規約

```
【{施設名}様】料金変動案_{YYYYMMDD}[_自動生成].xlsx
```

例:
- `【プールヴィラ古宇利島様】料金変動案_20251118.xlsx`
- `【アクアパレス北谷様】料金変動案_20260215_自動生成.xlsx`

施設名は `【...様】` から正規表現で抽出:
```
/【(.+?)様】/
```

## シート名

**固定**: `横カレンダー`

## シート構造

### 実際のレイアウト

```
      A     B                          C          D          E          ...  (→ 右端)
  1   ...   ...                        (offset)   (offset)   (offset)   ...
  2   ...   ...                        2025年11月 2025年11月 2025年11月 ...
  3   ...   (B3=日付)                   C3=日付1   D3=日付2   E3=日付3   ... (→ 空白で終端)
  4   1     部屋タイプ1                  H          G          P         ...
  5   2     部屋タイプ2                  H          G          P         ...
  6   3     部屋タイプ3                  Q          Q          G         ...
  7   4     部屋タイプ4                  Q          Q          G         ...
  8   5     0 (プレースホルダ)            (空)       (空)       (空)      ...
  :   ...   ...
  N   ...   (空白/None) → 下端終端
```

### ヘッダー

| 要素 | 位置 | 方向 |
|------|------|------|
| **日付** | Row 3, C3 から右方向 | 水平 (横) |
| **部屋タイプ名** | Column B, B4 から下方向 | 垂直 (縦) |

### データ領域

| 要素 | 値 |
|------|-----|
| **起点** | C4 (日付×部屋タイプの交差点) |
| **内容** | ランクコード（アルファベット 1 文字: A-Z） |
| **空セル** | 当該日付にランク未設定を意味する（エラーではない） |

## 起点: C4

- **固定**: 常に C4 セルからデータ取得を開始する
- C4 は最初の部屋タイプ × 最初の日付の交差セル
- C4 自体が空白の場合もある（当該期間のランク未設定）

## 右側終端判定

1. **Row 3（日付行）** を C3 から右方向に走査
2. セル値が `None` / 空文字 / 日付として解釈不可 のセルに到達したら終端
3. 終端セルの **1 つ前の列** が右端

### 注意点
- Row 3 には `datetime` オブジェクトが入っている（openpyxl が自動変換）
- Row 1 にはオフセット数値、Row 2 には月名文字列が入る場合がある（無視）

## 下側終端判定

1. **Column B（部屋タイプ列）** を B4 から下方向に走査
2. 以下の条件で終端と判定:
   - セル値が `None`
   - セル値が数値（`int`, `float`）→ プレースホルダ行（例: `0`）
   - セル値が空文字 / 空白のみ
3. 終端セルの **1 つ前の行** が下端

### 注意点
- 実際のファイルでは B8〜B13 に `0` が入っているが、部屋タイプ名ではない
- 有効な部屋タイプ名は **非空の文字列** であること
- 同じ部屋タイプ名が下方の別セクション（B16〜等）に繰り返し出現する場合があるが、
  最初のブロック（B4 から連続する有効行）のみをパース対象とする

## 矩形領域

終端判定の結果、パース対象は以下の矩形:

```
日付ヘッダー: C3 〜 {右端列}3
部屋タイプ:   B4 〜 B{下端行}
データ領域:   C4 〜 {右端列}{下端行}
```

## 実例（プールヴィラ古宇利島）

```
日付:       304 列 (2025-11-01 → 2026-08-31)
部屋タイプ: 4 行
  - ラグジュアリープールヴィラ(単泊)
  - ラグジュアリープールヴィラ(連泊)
  - with SAUNA(単泊)
  - with SAUNA(連泊)
データ:     1,216 セル (838 filled, 378 blank)
```
